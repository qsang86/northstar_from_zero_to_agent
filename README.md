# NorthStar 워크샵: From Zero to Agent

AI 에이전트를 위한 엔드투엔드 데이터 파이프라인 구축 워크샵 자료입니다.

## 파일 다운로드

| 파일 | 설명 | 다운로드 |
|------|------|---------|
| `setup.sql` | Snowflake 환경 설정 스크립트 (역할, 웨어하우스, 테이블 생성 및 데이터 로드) | [다운로드](https://github.com/qsang86/northstar_from_zero_to_agent/raw/main/setup.sql) |
| `marketing_data.csv` | 마케팅 캠페인 데이터 (추가 실습용) | [다운로드](https://github.com/qsang86/northstar_from_zero_to_agent/raw/main/marketing_data.csv) |

## 사용 방법

1. **setup.sql**: Snowsight에서 SQL Worksheet를 열고, `setup.sql` 내용을 붙여넣은 후 **Run All** (Ctrl+Shift+Enter)을 실행합니다.
2. **marketing_data.csv**: 로컬에 다운로드 후 Snowflake 테이블에 업로드하여 사용합니다.
